# Agent OS Thread Orchestration Kit

A lightweight architecture packet for designing agent systems as thread processes with sensors, plugins, routing, and evidence handoffs.

## Why this exists
Inspired by: https://x.com/nickbaumann_/status/2067396086252372465

## What it does
- Provides a first-trial packet for one real workflow.
- Includes an infographic explaining the operating model.
- Keeps a draft skill separate until evidence proves reuse.

## Quickstart
Open prototype/Trial Operator Handoff Checklist.md first, fill prototype/Agent OS Thread Map.md during a real orchestration task, map proof in prototype/Trial Evidence Index.md, grade proof with prototype/Agent OS Evidence Quality Rubric.md, fill prototype/Claim-to-Patch Ledger.md, then complete the promotion card and debrief before changing claims.
